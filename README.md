# Markdown Previewer

Preview Markdown instantly, on the web.

## Prerequisite

- [Node.js V20](https://nodejs.org/)
- [Yarn v1](https://yarnpkg.org/)

## Quick start

## Quick Start

Install dependencies:

```sh
yarn install
```

Start development server:

```sh
yarn dev
```

Linting and Styling:

```sh
yarn lint
yarn format
yarn format-check
```

Build production code to `./dist`:

```sh
yarn build
```
